Spice Box - プライバシーポリシー

最終更新日：2026年2月

1. 収集するデータについて

Spice Box は個人情報を一切収集しません。入力したスパイス情報、レシピ、写真、料理ログはすべてお使いのデバイス上にのみ保存されます。

2. ローカルストレージ

すべてのアプリデータは、ローカルデータベース（Isar）を使用してお使いのデバイス上にのみ保存されます。外部サーバーへの送信は一切行われません。

3. 写真について

アプリ内で撮影・選択した写真はお使いのデバイス上にのみ保存されます。お客様が明示的にシェア操作を行わない限り、アップロードや共有は行われません。

4. サードパーティサービス

現在、当アプリはログ収集や分析などのサードパーティ製SDKやトラッキングツールを一切使用していません。

5. お問い合わせ

プライバシーポリシーに関するご質問は、App Storeのレビューページよりご連絡ください。
